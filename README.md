# Student Performance Dashboard

## Project Summary

The Student Performance Dashboard is a Python application that manages student records and calculates academic performance. The project includes both a console-based application and a graphical user interface (GUI) built using Tkinter.

## Features

* Add and store multiple student records
* Input student details such as name, roll number, age, and gender
* Enter marks for five subjects
* Calculate total marks and percentage
* Assign grades automatically
* Search for a student by roll number
* Display the highest scorer
* Calculate average marks for each subject
* Validate user input and handle errors
* Simple GUI for entering and viewing student information

## How to Run the Project

### Console Version

1. Open the project folder in VS Code.
2. Open the terminal.
3. Run the command:
   python main.py

### GUI Version

1. Open the project folder in VS Code.
2. Open the terminal.
3. Run the command:
   python gui.py
